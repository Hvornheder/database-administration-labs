# USF Database Design & Administration — Lab Portfolio

Hands-on labs completed as part of ISM 4212: Database Design & Administration
at the University of South Florida (Fall 2024).

Labs cover cloud infrastructure provisioning, relational database administration,
SQL transactions, and managed cloud database services using AWS.

## Core Tools Used

- AWS EC2
- AWS RDS
- MySQL / MySQL Workbench
- Windows Server 2022
- Remote Desktop Protocol (RDP)
- VPC Security Groups

---

## Skills Demonstrated

- Cloud virtual machine provisioning and configuration
- VPC security group and firewall rule management
- Remote server administration via RDP
- Managed database service provisioning
- Cloud endpoint connectivity and remote database administration
- Relational schema creation and SQL execution in a cloud environment
